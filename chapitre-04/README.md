# Chapitre 4 — La Réalité : comprendre la complexité

> *"Kubernetes n'est pas complexe parce que les créateurs étaient sadiques. Il est complexe parce que le monde réel l'est."*

## Vue d'ensemble du chapitre

Après la prise de conscience vient l'acceptation. Ce chapitre explique pourquoi Kubernetes doit être complexe et pourquoi cette complexité est non seulement nécessaire, mais belle dans sa conception.
